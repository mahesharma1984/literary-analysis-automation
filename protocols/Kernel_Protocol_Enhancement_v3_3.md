# KERNEL PROTOCOL ENHANCEMENT - CONDENSED

## Stage 2B: Micro Device Identification

Identify 15-20 micro literary devices with quoted examples from the text.

**Requirements:**
- Minimum 15 devices (target: 20+)
- Include diverse device types (narrative, figurative, structural, rhetorical)
- Provide exact quoted examples (10-30 words each)
- Include chapter/page location for each example
- Brief analysis of each device's function

**Output Format:**
```json
{
  "devices": [
    {
      "device_name": "Device Name",
      "classification": "B|Me|V",
      "examples": [
        {
          "quote_snippet": "exact quoted text",
          "analysis": "brief explanation",
          "location": "Chapter X, page Y"
        }
      ]
    }
  ]
}
```

Select devices that are pedagogically relevant and clearly demonstrable in the text.
